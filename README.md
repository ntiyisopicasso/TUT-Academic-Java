# TUT Academic Java 🚀

Welcome to my academic repository! This repository serves as a centralized portfolio for the programmatic solutions, lab exercises, and practical challenges completed throughout my Information Technology / Computer Science diploma at the Tshwane University of Technology (TUT).

## 🛡️ Focus on Secure Coding
As an aspiring defensive cybersecurity professional, my primary objective with every project in this repository is to implement robust, vulnerability-free logic. Every script is engineered with a "security-first" mindset, focusing on:
* **Input Sanitization:** Mitigating unexpected errors by trimming whitespace and enforcing strict case consistency.
* **Defensive Exception Handling:** Preventing runtime application crashes through structured `try-catch` blocks (e.g., handling `NumberFormatException`).
* **Boundary Validation:** Enforcing strict data-type and length constraints to prevent logical injection or overflow bugs.

## 📂 Repository Structure
The codebase is systematically organized by academic milestones and concepts:
* `/Core_Strings` - Dynamic string parsing and substrings (e.g., Secure Institutional Email Validator).
* `/Data_Structures` - Arrays, ArrayLists, and data organization models.
* `/Input_Output` - High-performance data streaming using `BufferedReader`.

## 🛠️ Tech Stack & Tools
* **Language:** Java SE
* **Environment:** OpenJDK / IntelliJ IDEA / Eclipse
* **Focus Areas:** Object-Oriented Programming (OOP), Logic Building, Secure Development
